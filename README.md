A simple web-based tool that randomly assigns Mafia game characters to players. Perfect for quickly setting up a Mafia game without manual role distribution.

🚀 Features
👥 Add any number of players
📝 Enter player names dynamically
🎭 Select which roles to include (Mafia, Civilian, etc.)
🎲 Random role assignment
⚡ Fast and easy to use
📱 Responsive design
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
